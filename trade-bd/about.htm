Hello There,

Thank you for your kind interest about me. I'm the only developer of this web site. The history behind developing this web site is basically my and my friends' needs. I had been feeling for a web site like this one for a long time (almost 2 years). But unfortunately there was no web site made specially for Bangladeshies for trading old/new things. I worked almost 200 hours for the development of this site. I used Active Server Pages (using VBScript) as the front end language and SQL as the back end. I used Photoshop for graphics.
